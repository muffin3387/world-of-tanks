# World of Tanks - Complete Game Implementation

完全なWorld of Tanksゲーム実装です。バックエンドサーバー、フロントエンドUI、リアルタイム戦闘システムを含みます。

## 機能

### ゲームシステム
- **プレイヤー管理**: レベル、資金、経験値システム
- **戦車管理**: 複数国の戦車ツリー（ドイツ、ソ連、アメリカ）
- **強化システム**: 砲身、車体、エンジンの3段階強化
- **ショップシステム**: 毎日変わるプレミアム戦車販売

### 戦闘システム
- **リアルタイムバトル**: 複数の敵と味方が参加する戦闘
- **AI敵**: パトロール・索敵・交戦の3状態を持つAI
- **動的マップ**: 砂漠、雪、沿岸地形の3種類
- **天候システム**: 晴れ、霧、雨、雪の4種類（照準精度に影響）
- **ミニマップ**: リアルタイム戦況表示
- **ダメージシステム**: 装甲角度によるダメージ計算

### ゲームメカニクス
- **操作**: WASD移動、マウス照準、クリック射撃
- **武装**:
  - 主砲：弾薬数制限、リロード時間
  - 機銃：高速射撃、弾薬数制限（一部戦車のみ）
- **ユーティリティ**:
  - 修復キット：HP回復（クールタイム180秒）
  - 機雷配置：敵を撃破（クールタイム300秒）
- **障害物**: 岩、木箱、壁、バリケード（木箱は破壊可能）
- **エフェクト**: 被弾時フラッシュ、砲口炎、爆発、煙、火

### 戦車スペック

#### ドイツ
- **Pz.Kpfw. II** (Tier 1): 軽戦車、機動力特化
- **Pz.Kpfw. III** (Tier 2): 中戦車、バランス型
- **Tiger I** (Tier 3): 重戦車、高火力・高装甲

#### ソ連
- **T-26** (Tier 1): 軽戦車、高速
- **T-34** (Tier 2): 中戦車、革新的傾斜装甲
- **IS-2** (Tier 3): 重戦車、最強クラス

#### アメリカ
- **M4 Sherman** (Tier 1): 中戦車、信頼性高
- **M26 Pershing** (Tier 2): 重巡戦車、高火力
- **T34 American** (Tier 3): 重戦車、バランス型

#### プレミアム戦車（ショップ）
- **Jagdtiger**: ドイツ駆逐戦車、怪物火力
- **ISU-152**: ソビエト駆逐戦車、榴弾砲搭載
- **T95**: アメリカ自走砲、最強装甲

## インストール

### 必要環境
- Node.js 14以上
- npm または yarn

### セットアップ

1. **依存関係をインストール**
```bash
npm install
```

2. **サーバーを起動**
```bash
npm start
```

3. **ブラウザでアクセス**
```
http://localhost:3000
```

開発モード（ファイル変更時に自動リロード）:
```bash
npm run dev
```

## ゲームプレイ

### メニュー画面
- **戦闘開始**: 戦闘ステージへ移動
- **ガレージ**: 戦車の選択と強化
- **ショップ**: 新しい戦車の購入
- **初期化**: ゲームをリセット

### ガレージ画面
1. 国を選択（ドイツ/ソ連/アメリカ）
2. 戦車をクリックして選択
3. 強化ボタンで能力を向上（各3,500資金）
4. 「この戦車で出撃する」をクリック

### 戦闘画面

#### 操作
- **W/↑**: 前進
- **S/↓**: 後退
- **A/←**: 左旋回
- **D/→**: 右旋回
- **マウス移動**: 砲塔照準
- **左クリック**: 主砲発射
- **スペース**: 機銃発射
- **R**: 修復キット使用
- **M**: 機雷配置
- **ESC**: 一時停止

#### 戦闘メカニクス
- HP が0になると戦闘不能
- 敵を撃破して資金と経験値を獲得
- 味方の支援を受ける（全員を失うと敗北）
- 装甲角度でダメージが変わる
- 射撃すると照準精度が低下

### ショップ画面
- **本日の一機**: 毎日異なるプレミアム戦車（150,000資金）
- **開発ツリー**: 各国の戦車を段階的に購入
  - Tier 2: レベル5、25,000資金
  - Tier 3: レベル10、35,000資金

## API エンドポイント

### プレイヤー管理
- `POST /api/player/init` - 新規プレイヤー初期化
- `GET /api/player/:playerId` - プレイヤー情報取得
- `POST /api/player/:playerId/reward` - 報酬授与

### 戦車管理
- `POST /api/tank/upgrade` - 戦車強化
- `POST /api/tank/buy` - 開発ツリーから購入
- `POST /api/tank/buy-shop` - ショップから購入
- `POST /api/battle/select-tank` - 戦闘用戦車選択

### ショップ
- `GET /api/shop/today` - 本日のプレミアム戦車

### ゲーム
- `POST /api/game/reset` - ゲームリセット
